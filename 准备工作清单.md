## A. 开发环境

### A1. Python 虚拟环境

- [ ] 确认 Conda 可用：`conda --version`
- [ ] 创建环境：`conda create -n job-hunt python=3.11 -y`
- [ ] 激活环境：`conda activate job-hunt`
- [ ] 验证版本：`python --version`（建议 3.11.x）

> 你当前系统 Python 为 3.12.7，**建议仍用 3.11 建 conda 环境**，LangChain / Chroma 兼容性更稳。

### A2. 安装依赖

- [ ] 进入仓库目录：`cd "d:\job hunting"`
- [ ] 安装：`pip install -r requirements.txt`
- [ ] 若 chromadb / ragas 安装失败，见下方「常见问题」

### A3. 冒烟测试

- [ ] 复制 `.env.example` → `.env`，填入 API Key
- [ ] 运行：`python scripts/hello_llm.py`
- [ ] 看到模型回复即表示 API + 环境 OK

### A4. 开发工具（检查即可）

- [ ] Cursor 已登录，可正常使用 Agent
- [ ] Git 已配置用户名邮箱（`git config user.name` / `user.email`）
- [ ] 终端默认进入项目目录（可选，方便日常）

## 常见问题

| 问题 | 处理 |
|------|------|
| `pip install chromadb` 失败 | 先 `pip install chromadb==0.5.23`，或 Windows 上装 VS Build Tools |
| 无 API 回复 / 401 | 检查 `.env` 路径、Key 是否有多余空格 |
| conda activate 无效 | PowerShell 执行 `conda init powershell` 后重开终端 |
| CUDA 不可用 | RAG/Agent 不依赖 GPU；微调走 Colab |
| 依赖安装太慢 | `pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple` |

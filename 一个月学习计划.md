## 三、每日 8 小时作息模板

> 离职全职学习需可持续，**不要 8h 全坐 dead learning**；下面含番茄休息。

| 时间 | 时长 | 块 | 内容 |
|------|------|-----|------|
| 09:00 - 10:30 | 1.5 h | **块 1 · 输入** | 看课程 / 读官方文档，做结构化笔记 |
| 10:30 - 10:45 | 15 min | 休息 | 离开屏幕 |
| 10:45 - 12:15 | 1.5 h | **块 2 · 跟练** | 跟着课程写代码，禁止复制粘贴不思考 |
| 12:15 - 14:00 | — | 午饭 + 午休 | |
| 14:00 - 15:30 | 1.5 h | **块 3 · 项目** | 当日主项目功能开发 |
| 15:30 - 15:45 | 15 min | 休息 | |
| 15:45 - 17:15 | 1.5 h | **块 4 · 加深** | 扩展练习 / 调试 / 写测试 |
| 17:15 - 18:00 | 45 min | **块 5 · 输出** | Git commit、学习日志、Checklist、口述复盘 |

**合计：** 7.5 h 有效学习 + 0.5 h 休息 ≈ **8 h**

**每周日下午块 5 延长：** 周复盘 + README 交付（至 19:00）。
